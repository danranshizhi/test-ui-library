# my-ui-library

examples文件夹用作实例demo

lib文件夹用于存放打包生成的文件
packages文件夹用于编写存放的组件

  每个组件建一个文件夹，该文件夹中包含src文件夹及最外层index.js文件--对外提供对组件的引用

```
npm install
```
